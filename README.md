# Samu-Gym

Mini web app per visualizzare la scheda palestra su Fire Stick / TV.

## File
- `index.html` = pagina principale
- `style.css` = grafica
- `app.js` = logica timer
- `workout.json` = scheda modificabile
- `img/` = immagini esercizi

## Come provarla sul PC
Apri `index.html` con Chrome.

## Come cambiare scheda
Modifica solo `workout.json`.
Puoi cambiare:
- nome esercizio
- gruppo
- ripetizioni
- recupero
- durata per esercizi a tempo
- immagine
- consigli tecnici

## Comandi
- FATTO / AVANTI
- PAUSA
- INDIETRO
- Tastiera: Invio/Spazio/Freccia destra = avanti, P = pausa, Freccia sinistra = indietro

## Fire Stick
La strada consigliata è pubblicare la cartella su GitHub Pages e aprire il link con Silk Browser.
